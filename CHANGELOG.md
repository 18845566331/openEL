# 更新日志

## v1.1.0 (2026-03-17)

### 🆕 新功能

#### 缺陷分类等级判定系统
- 新增每张图片的 A/B/C 类别判定功能，基于可配置的缺陷等级表
- 当一张图片出现多个缺陷时，按照等级优先级（1-12级）取最严重类别判定
- A/B/C 类别优先于 1-12 等级排序（如同为 C 类则按等级高低区分）
- 文件列表中显示每张图片的 A/B/C 等级判定结果

#### 实时统计增强
- 检测统计面板新增 A/B/C 类数量和百分比实时统计
- 右上角"等级统计"改名为"类别统计"

#### 报告导出增强
- 报告中自动填充 `{组件型号}` — 支持多组件型号（逗号连接）
- 报告中自动填充 `{YYYY/MM/DD~YYYY/MM/DD}` 检测日期
- 报告中自动填充 `{A类组件数量}` `{B类组件数量}` `{C类组件数量}` 及对应比例
- 报告中自动填充 `{被测组件数量}`
- 动态生成缺陷类别明细行：检测到几种缺陷就添加几行，包括各类别的 A/B/C 数量、总数量和占比
- 修复合并单元格导致数据重复写入的问题
- 修复 Word 模板中占位符被空格拆分导致匹配失败的问题
- 修复全局占位符替换抢先覆盖导致 Section 5 无法填充的问题

### ⚡ 优化

#### 缺陷等级配置 UI 优化
- 缺陷名称改为可编辑输入框，支持自定义缺陷名称
- 等级编号改为可编辑，支持手动调整等级
- 新增排序功能（↑↓ 箭头），支持拖拽调整缺陷排序
- 新增缺陷类别后可直接修改名称
- 整体显示增大：字体 10px → 12px，输入框宽度 36px → 44px
- 删除缺陷后自动重新编号

#### 参数设置面板优化
- 检测参数和标注框显示区域布局更紧凑
- 开关控件使用 VisualDensity.compact 减少间距
- 为缺陷等级配置区域留出更多显示空间

### 🐛 修复

#### Word 模板处理
- 项目信息中"尽调周期"标签改为"检测日期"
- 修复 `{组件型号}` 从 pv_modules 列表提取型号的逻辑
- 修复 Section 3b 空格归一化回退扫描误替换 Section 5 专用占位符的问题
- 新增热补丁部署机制支持：代码修改后无需重新编译 exe
