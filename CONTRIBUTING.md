# 参与贡献 (Contributing)

感谢你对 **EL光伏组件缺陷检测系统** 进行贡献的兴趣！

我们非常欢迎以各种方式向本项目做出贡献，包括但不限于：
1. 提交 Bug 报告。
2. 添加新的检测模型或算法优化。
3. 扩展前端界面的功能。
4. 修正拼写错误和增强文档。

## 提交反馈和 Issue
如果系统出现了意外错误，请不要犹豫，在 GitHub 上向我们发起 Issue，尽量提供你的设备系统，以及相应的出错截图或日志。如果你对我们的**无人机红外巡检去恶心全自动流程**或**光伏组件自动巡检**这些特性有独到的提议，也非常欢迎你在 Issue 进行探讨！

## 提交代码 (Pull Request)
在提交代码前，请确保：
1. 本地完成核心功能测试 (Python 端使用 `pytest`，Flutter 端运行 `flutter test`)。
2. 保持代码风格统一。
3. 如果是在引入全新的依赖包，请在 PR 中附上理由并确保第三方包符合我们 GPLv3 的版权规约。

## 开发路线图
参考 README.md 所述的开发路线，目前我们优先采纳针对于自动巡检拓展的合并请求。

感谢你与我们一起将此项目变得更好！
