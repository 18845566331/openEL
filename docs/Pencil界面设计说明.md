# Pencil 界面设计说明（中文）

已在 Pencil 中完成 `EL 光伏组件缺陷检测系统` 工作台原型设计，核心结构：

1. 左侧深色导航栏：实时检测 / 批量任务 / 模型管理 / 检测报告 / 系统设置
2. 中央主工作区：控制按钮、阈值标签、图像预览区
3. 右侧数据面板：不良组件、一次通过率、缺陷分类、最近文件、运行日志

主视觉参数：

1. 主色：`#1D4ED8`
2. 背景：`#F5F7FB`
3. 侧边栏：`#0F172A`
4. 卡片圆角：`14-16px`

界面已全部中文化，可与 Flutter 页面 `frontend/lib/pages/workbench_page.dart` 一一对应。
